## Getting Started

### Setup

To setup the project locally, clone the repository using:

```
```

### Install dependencies
run:
```
$ bundle install
```

### Setup the database
run:
```
$ rails db:create
$ rails db:migrate
```

### Run the app
run:
```
$ ./bin/dev and follow the instructions 
```
